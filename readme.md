# 🌿 RenewYou — Wellness Subscription Box

> *Renew your mind. Renew your life.*

RenewYou is a customized wellness subscription box service that delivers curated mental wellness tools — including journals, essential oils, and books — right to your door. Built as a full e-commerce web application focused on promoting mental health and self-care.

---

## 📌 Project Overview

| Field | Details |
|---|---|
| **Industry** | Mental Health / E-commerce |
| **Type** | College Assignment Project |
| **Platform** | Web Application |
| **Tech Stack** | HTML, CSS, JavaScript |

**Problem Statement:**
Many individuals struggle to find the right mental wellness resources. RenewYou solves this by offering a personalized subscription model that curates and delivers wellness products tailored to each user's needs — making self-care accessible, consistent, and meaningful.

---

## ✨ Features

- 🛍️ **Subscription Box Plans** — Choose from different subscription tiers suited to your wellness goals
- 🧘 **Curated Product Selection** — Each box includes hand-picked items like journals, aromatherapy oils, and wellness books
- 🎯 **Personalization** — Boxes tailored to the subscriber's mental health preferences and goals
- 🛒 **E-Commerce Flow** — Browse products, select a plan, and complete checkout seamlessly
- 📦 **Box Preview** — See what's inside before you subscribe
- 📱 **Responsive Design** — Works across desktop, tablet, and mobile devices
- 🌐 **Clean UI** — Calming, wellness-oriented design for an immersive user experience

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Edge, Safari)
- No installations required

### Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/Kv-404/RenewYou.git
   ```

2. **Navigate to the project folder**
   ```bash
   cd RenewYou
   ```

3. **Open in browser**
   ```
   Open index.html in your preferred browser
   ```

## 🎯 Use Case

RenewYou targets individuals seeking consistent mental wellness support. By removing the friction of researching and buying wellness products individually, RenewYou delivers a thoughtfully assembled box that helps users build healthy habits month after month.
